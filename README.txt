How to run the ipynb files:

1. Download (publicly available) dataset from https://www.kaggle.com/kmader/skin-cancer-mnist-ham10000

2. Upload zipped folder of the downloaded dataset to Google Drive.

3. Load either one of the .ipynb files in Google Colab.

4. Change path of "!unzip" according to your folder location. Pick a destination for the unzipped files.

5. Change path to read unzipped files under "Read images in dataset".

6. Run the necessary codes.
